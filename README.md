# 🔒 تطبيق الدردشة السرية (E2EE Secret Chat App)

تطبيق دردشة آمن جداً يعتمد على التشفير من طرف إلى طرف (End-to-End Encryption). لا يتم حفظ الرسائل في أي قاعدة بيانات، ولا يرى الخادم (Server) سوى نصوص مشفرة لا يمكن فكها إلا باستخدام المفتاح السري الموجود لدى المستخدمين في نفس الغرفة.

## 🌟 المميزات
- تشفير AES من طرف إلى طرف (باستخدام Crypto-js).
- لا توجد قواعد بيانات (Zero Data Retention).
- واجهة مستخدم داكنة وجذابة تدعم اللغة العربية.
- اتصال لحظي وسريع جداً عبر Socket.io.
